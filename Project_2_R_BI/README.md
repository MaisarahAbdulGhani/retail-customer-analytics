## Customer Churn and Segmentation Analysis

### Objective
Identify churn risk and segment customers based on purchasing behaviour.

### Tools
- Python, Pandas, Numpy
- R Programming
- Tableau / BI Dashboards

### Key Analysis
- RFM analysis
- K-means clustering
- Chi-square and ANOVA tests

### Outcome
Identified loyal, at-risk, and churned customer segments for targeted strategies.
